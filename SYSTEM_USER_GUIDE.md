# 智能课程表系统使用指南

## 1. 系统概述

智能课程表系统是一款基于微信小程序的课程表管理工具，旨在为用户提供便捷、高效的课程表管理和共享功能。系统支持课程信息录入、可视化课表展示、课表共享与社区、信息完善机制、质量评价体系以及认证验证机制等功能，满足用户多样化的课程表管理需求。

### 1.1 系统特点
- **直观的课程录入**：支持手动录入和文件导入两种方式
- **可视化课表展示**：提供周视图和日视图，支持视图切换
- **课表共享与社区**：生成唯一课表ID，支持课表导入和分享
- **信息完善机制**：引导用户完善课程详细信息，提供信息模板
- **质量评价体系**：统计课表热度，支持用户评价和星级评分
- **认证验证机制**：官方认证和班级绑定，确保课表质量

## 2. 功能模块介绍

### 2.1 课表管理功能

#### 2.1.1 课程信息录入
- **手动录入**：通过直观的表单界面，填写课程名称、时间、地点、教师等信息
- **文件导入**：支持Excel和JSON格式文件解析，提供文件格式模板和导入指引

#### 2.1.2 可视化课表展示
- **周视图**：以日历形式展示整周课程安排
- **日视图**：按时间段详细展示单日课程
- **视图切换**：提供便捷的视图切换控件，记忆用户偏好设置

### 2.2 课表共享与社区功能

#### 2.2.1 课表分享机制
- 为每个课表生成唯一不可重复的ID编码
- 实现分享功能，支持通过ID码进行课表导入

#### 2.2.2 课表广场
- 构建社区平台，展示可共享的课表资源
- 按热度排行展示课表，方便用户发现优质课表

### 2.3 课表信息完善机制

- 在课表发布流程中增加信息完善引导
- 设置课程教师、教室、课程简介等信息为推荐填写项
- 设计友好的提示机制，鼓励用户完善课程详细信息
- 提供信息模板，标准化课程信息填写格式

### 2.4 课表质量评价体系

#### 2.4.1 热度排行功能
- 统计并展示课表被导入次数
- 实现课表广场热度排行榜，按导入量降序排列

#### 2.4.2 用户评价系统
- 允许用户对导入的课表进行点赞操作
- 实现星级评分功能（1-5星）
- 展示课表的平均评分和评价数量，辅助用户筛选优质课表

### 2.5 课表认证与验证机制

#### 2.5.1 官方认证体系
- 设计官方认证标识，用于标记经过审核的优质课表
- 建立班级绑定机制，允许班长或课代表申请成为班级课表管理员
- 支持管理员发布班级标准版课表，带有特殊标识
- 实现标准版课表更新通知功能，确保用户获取最新课程信息

#### 2.5.2 错误反馈渠道
- 允许用户报告课表错误，由管理员核实修正
- 提供详细的错误类型选择，方便管理员快速定位问题

## 3. 使用指南

### 3.1 首次使用

1. **进入系统**：打开微信小程序，搜索并进入智能课程表系统
2. **授权登录**：根据提示授权小程序获取用户信息
3. **配置云开发环境**：在小程序设置中配置云开发环境ID（由管理员提供）

### 3.2 课程录入

#### 3.2.1 手动录入
1. 点击首页的"课程录入"导航入口
2. 选择"手动录入"选项
3. 填写课表名称
4. 点击"+ 添加课程"按钮，填写课程详细信息：
   - 课程名称
   - 教师姓名
   - 上课地点
   - 星期和时间
   - 周次
   - 课程简介（可选）
5. 重复步骤4添加更多课程
6. 点击"保存课表"按钮完成录入

#### 3.2.2 文件导入
1. 点击首页的"课程录入"导航入口
2. 选择"文件导入"选项
3. 下载文件模板（Excel或JSON格式）
4. 按照模板格式填写课程信息
5. 点击"选择文件"按钮，上传填写好的文件
6. 预览导入的课程信息，确认无误后点击"导入课程"
7. 填写课表名称，点击"保存课表"按钮完成录入

### 3.3 课表展示

1. 点击首页的"课表展示"导航入口
2. 在课表展示页面，您可以：
   - 切换周视图和日视图
   - 使用日期选择器查看不同周/日的课程
   - 点击课程卡片查看课程详细信息

### 3.4 课表共享

#### 3.4.1 分享课表
1. 在课表详情页面，点击"分享课表"按钮
2. 选择"复制课表ID"或"分享给好友"
3. 将课表ID分享给其他用户

#### 3.4.2 导入课表
1. 点击首页的"课表广场"导航入口
2. 在"导入课表"部分，输入收到的课表ID
3. 点击"导入"按钮
4. 预览课表信息，确认无误后完成导入

### 3.5 课表广场

1. 点击首页的"课表广场"导航入口
2. 浏览热门课表列表，查看课表的热度、评价等信息
3. 点击课表卡片查看详情
4. 点击"导入"按钮导入感兴趣的课表
5. 点击"分享"按钮分享课表给好友

### 3.6 评价课表

1. 在课表详情页面，找到"评价与反馈"部分
2. 点击"点赞"按钮为课表点赞
3. 点击星级评分组件为课表评分（1-5星）
4. 查看其他用户的评价和课表的平均评分

### 3.7 认证管理

1. 点击首页的"认证管理"导航入口

#### 3.7.1 班级绑定
1. 在"班级绑定"部分，填写班级名称和申请理由
2. 点击"申请班级管理员"按钮
3. 等待审核通过后，成为班级管理员

#### 3.7.2 管理班级课表
1. 在"我的班级"部分，找到已管理的班级
2. 点击"管理课表"按钮，发布和管理班级标准版课表
3. 点击"查看成员"按钮，查看班级成员列表

#### 3.7.3 官方认证
1. 在"官方认证"部分，点击"申请官方认证"按钮
2. 选择需要认证的课表
3. 等待审核通过后，课表将获得官方认证标识

#### 3.7.4 错误反馈
1. 在"错误反馈"部分，填写课表ID、错误类型和错误描述
2. 点击"提交反馈"按钮
3. 管理员将收到反馈并核实修正

## 4. 常见问题

### 4.1 课程录入相关

**Q: 如何导入Excel格式的课程表？**
A: 在课程录入页面选择"文件导入"选项，下载Excel模板，按照模板格式填写课程信息，然后上传文件即可。

**Q: 导入文件失败怎么办？**
A: 请检查文件格式是否正确，确保按照模板格式填写，并且文件大小不超过限制。如果问题仍然存在，请尝试使用手动录入方式。

### 4.2 课表展示相关

**Q: 如何切换周视图和日视图？**
A: 在课表展示页面，点击顶部的"周视图"或"日视图"选项卡即可切换。

**Q: 为什么课表没有显示课程？**
A: 请检查是否已经添加了课程，并且课程的时间设置正确。如果问题仍然存在，请尝试重新进入页面或刷新数据。

### 4.3 课表共享相关

**Q: 如何获取课表ID？**
A: 在课表详情页面，点击"分享课表"按钮，选择"复制课表ID"即可获取。

**Q: 导入课表失败怎么办？**
A: 请检查课表ID是否正确，确保课表ID没有输入错误。如果问题仍然存在，请联系分享课表的用户确认课表是否存在。

### 4.4 评价系统相关

**Q: 如何为课表评分？**
A: 在课表详情页面，找到"评价与反馈"部分，点击星级评分组件即可为课表评分（1-5星）。

**Q: 可以修改我的评价吗？**
A: 是的，您可以再次点击星级评分组件修改您的评价。

### 4.5 认证管理相关

**Q: 如何成为班级管理员？**
A: 在认证管理页面的"班级绑定"部分，填写班级名称和申请理由，点击"申请班级管理员"按钮，等待审核通过后即可成为班级管理员。

**Q: 如何发布班级标准版课表？**
A: 在认证管理页面的"我的班级"部分，找到已管理的班级，点击"管理课表"按钮，按照提示发布班级标准版课表。

**Q: 如何申请官方认证？**
A: 在认证管理页面的"官方认证"部分，点击"申请官方认证"按钮，选择需要认证的课表，等待审核通过后即可获得官方认证标识。

### 4.6 其他问题

**Q: 系统支持哪些设备？**
A: 系统基于微信小程序开发，支持所有安装了微信的移动设备，包括手机和平板电脑。

**Q: 如何联系客服？**
A: 如果您在使用过程中遇到问题，可以通过小程序内的客服功能联系我们，我们将尽快为您解决问题。

**Q: 系统会保存我的个人信息吗？**
A: 系统会保存必要的用户信息用于功能实现，但我们会严格保护您的个人隐私，不会将您的信息用于其他用途。
